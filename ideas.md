## Idea 1: "Cyberpunk Neon Luxe"

### Design Movement
Inspirace cyberpunk estetikou kombinovanou s luxury fashion branding - myslím na Blade Runner 2049 meets Gucci.

### Core Principles
1. **Kontrastní dualita** - Hluboká černá (#050508) vs. elektrické neonové akcenty
2. **Vertikální dramatičnost** - Vysoké sekce s dramatickými přechody
3. **Glitch estetika** - Jemné digitální artefakty jako designový prvek
4. **Asymetrická elegance** - Rozbití tradičních grid layoutů

### Color Philosophy
- **Primární**: Deep void black (#050508) - absolutní hloubka
- **Sekundární**: Electric amber (#FFB800) - energie a luxus
- **Terciární**: Cyber violet (#8B5CF6) - technologická mystika
- **Akcent**: Hot pink (#FF2D92) - urgency a CTA

Emocionální záměr: Pocit exkluzivity a futuristické síly

### Layout Paradigm
Broken grid s floating panels - karty "levitují" v prostoru s různými z-indexy. Hero sekce zabírá 100vh s diagonálním dělením.

### Signature Elements
1. **Neon glow trails** - Světelné stopy za pohybujícími se elementy
2. **Holografické karty** - Karty s rainbow reflection efektem při hoveru
3. **Scanline overlay** - Jemný CRT efekt přes celou stránku

### Interaction Philosophy
Každá interakce vyvolává "elektrický" feedback - ripple efekty, glow pulzy, micro-glitche.

### Animation
- Hero text: Typewriter efekt s cursor blink
- Karty: Float animation s parallax na scroll
- CTA: Pulse glow s neon flicker
- Transitions: Glitch wipe mezi sekcemi

### Typography System
- **Display**: "Orbitron" - futuristický, geometrický
- **Body**: "Space Grotesk" - čitelný, technický
- Hierarchie: 72px/48px/32px/18px/14px
</text>

## Idea 2: "Obsidian Alchemy"

### Design Movement
Dark academia meets alchemical mysticism - inspirace starými grimoires, ale s moderním minimalistickým přístupem.

### Core Principles
1. **Posvátná geometrie** - Kruhy, mandaly, zlatý řez jako strukturální základ
2. **Textured depth** - Vrstvy s jemnými texturami (papír, noise, grain)
3. **Rituální flow** - Uživatel je veden jako při rituálu transformace
4. **Tiché bohatství** - Luxus bez křiku, elegance v detailech

### Color Philosophy
- **Primární**: Obsidian (#0A0A0F) - hluboká tma s jemným modrým undertone
- **Sekundární**: Alchemical gold (#D4AF37) - skutečné zlato, ne neon
- **Terciární**: Aged parchment (#F5F0E6) - kontrast pro text
- **Akcent**: Blood ruby (#9B1B30) - urgency s elegancí

Emocionální záměr: Pocit transformace, tajemství a časem prověřené moudrosti

### Layout Paradigm
Centrální mandala jako navigační hub - 7 kategorií jako 7 bodů kruhu. Scroll reveal odhaluje vrstvy jako otevírání staré knihy.

### Signature Elements
1. **Zlaté linky** - Tenké zlaté čáry spojující sekce jako alchemické diagramy
2. **Mandala navigace** - Interaktivní kruhový menu v hero sekci
3. **Parchment cards** - Karty s jemnou papírovou texturou a burnt edges

### Interaction Philosophy
Pomalé, deliberate animace - každá akce má váhu a význam. Hover efekty odhalují skryté informace.

### Animation
- Hero: Mandala se pomalu otáčí, body svítí při hover
- Scroll: Parallax vrstvy s různou rychlostí
- Cards: Subtle lift s golden glow
- Page transitions: Fade through black s golden particles

### Typography System
- **Display**: "Cinzel" - elegantní serif s římským charakterem
- **Body**: "Cormorant Garamond" - klasický, čitelný
- Hierarchie: 64px/40px/28px/18px/14px

## Idea 3: "Neural Constellation"

### Design Movement
Biomechanická estetika inspirovaná neuronálními sítěmi a hvězdnými mapami - spojení organického s technologickým.

### Core Principles
1. **Organická konektivita** - Vše je propojeno viditelnými "synapsy"
2. **Breathing interfaces** - UI dýchá, pulzuje, žije
3. **Infinite canvas** - Pocit nekonečného prostoru a možností
4. **Data as art** - Statistiky a čísla jako vizuální prvky

### Color Philosophy
- **Primární**: Deep space (#0D0D12) - kosmická tma
- **Sekundární**: Neural amber (#F59E0B) - energie synapsí
- **Terciární**: Plasma blue (#3B82F6) - technologická chladnost
- **Akcent**: Quantum green (#10B981) - growth a success

Emocionální záměr: Pocit být součástí něčeho většího, inteligentního systému

### Layout Paradigm
Node-based layout - každá sekce je "uzel" propojený animovanými čarami. Hero obsahuje interaktivní síť kde uzly reprezentují kategorie.

### Signature Elements
1. **Synapse connections** - Animované čáry mezi elementy
2. **Floating orbs** - Světelné koule reprezentující iBoty
3. **Data particles** - ROI čísla jako plovoucí částice

### Interaction Philosophy
Každý element reaguje na proximity - přiblížení myši aktivuje okolní uzly. Kliknutí vytváří ripple efekt přes síť.

### Animation
- Hero: Síť uzlů pulzuje a reaguje na mouse movement
- Stats: Čísla se "počítají" při scroll into view
- Cards: Hover aktivuje connection lines k sousedním kartám
- Background: Jemný particle flow jako neuronální aktivita

### Typography System
- **Display**: "Syne" - geometrický, moderní, unikátní
- **Body**: "Inter" s variable weight - čistý, technický
- Hierarchie: 56px/36px/24px/16px/14px

## Vybraný přístup: Idea 2 - "Obsidian Alchemy"

Vybírám tento přístup protože:
1. **Unikátnost** - Vyhýbá se typickému "AI startup" looku
2. **Hormozi alignment** - Rituální flow podporuje konverzní cestu
3. **Premium feel** - Skutečné zlato (#D4AF37) působí luxusněji než neonové barvy
4. **Mandala navigace** - Perfektně mapuje 7 kategorií iBotů
5. **Textury** - Přidávají hloubku bez přehlcení

### Implementační poznámky
- Background: #0A0A0F s jemným noise overlay
- Gold akcenty: #D4AF37 pro premium feel, #F59E0B pro CTA
- Font: Cinzel pro headlines, Cormorant Garamond pro body
- Mandala: SVG s CSS animacemi pro interaktivitu
- Cards: Subtle paper texture s golden border na hover
